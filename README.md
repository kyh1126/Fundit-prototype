# Fundit - Web3 보험 크라우드펀딩 플랫폼

## 프로젝트 개요
Fundit은 Web3 기술을 활용한 혁신적인 보험 크라우드펀딩 플랫폼입니다. 이 플랫폼은 보험 상품의 제안, 입찰, 계약 체결, 청구 처리까지의 전 과정을 블록체인 상에서 투명하게 관리합니다.

## 주요 기능

### 1. 리뷰 시스템
- 계약 소유자의 리뷰 제출 및 관리
- 리뷰 수정 및 삭제 기능
- 리뷰 신고 및 처리 시스템
- 리뷰 품질에 따른 차등 보상

### 2. 보험 계약 시스템
- 제안 생성 및 관리
- 보험사 등록 및 입찰
- 계약 체결 및 상태 관리

### 3. 청구 시스템
- 청구 제출 및 증빙서류 관리
- 오라클 기반 검증 시스템
- 보험금 자동 지급

### 4. 토큰 시스템
- 토큰 발행 및 전송
- 리뷰 보상 시스템
- 보상 분배 이벤트

## 기술 스택
- Solidity: 스마트 계약 개발
- Hardhat: 개발 환경 및 테스트
- TypeScript: 테스트 코드 작성
- OpenZeppelin: 스마트 계약 라이브러리

## 설치 및 실행
1. 저장소 클론
```bash
git clone https://github.com/your-username/fundit.git
cd fundit
```

2. 의존성 설치
```bash
npm install
```

3. 테스트 실행
```bash
npx hardhat test
```
